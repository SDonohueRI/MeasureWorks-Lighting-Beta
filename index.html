Accepts copied cell ranges from Excel or Google Sheets (tab-separated) and CSV text. A header row is auto-detected and columns auto-mapped by name — e.g. Space, Space Type, Existing Fixture, Qty, Proposed Fixture, Hours, Cost. Fixture text is matched against the wattage table; unmatched fixtures import with 0 W for manual entry. You can adjust any column's mapping before importing.
